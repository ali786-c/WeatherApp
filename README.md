# WeatherApp

WeatherApp is a simple Android application built entirely in **Kotlin**, providing users with real-time weather information for their location or selected cities. This app demonstrates best practices in Android development, clean architecture, and modern UI design.

## Features

- Current weather updates for any city
- Location-based weather detection
- Intuitive, responsive user interface
- Displays temperature, humidity, wind speed, and weather conditions
- Simple search functionality
- Error handling and network connectivity checks

## Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio)
- Android device or emulator with API level 21+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ali786-c/WeatherApp.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.

## Usage

- Launch the app to view weather information for your current location.
- Use the search bar to view weather for other cities.

## Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/) (100%)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependencies**:
  - Retrofit (network calls)
  - Coroutine (asynchronous operations)
  - LiveData & ViewModel
  - Glide/Picasso (image loading)
  - Room (optional, for local caching)
